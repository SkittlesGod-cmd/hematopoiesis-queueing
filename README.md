# hematopoiesis-queueing

Semi-Markovian queueing network inference of rate-limiting bottlenecks in hematopoietic stem cell differentiation from single-cell transcriptomic time-series.

## Project structure

```
├── data/               # Raw, processed, and synthetic data
├── src/queuediff/      # Installable Python package
├── scripts/            # Pipeline orchestration scripts
├── notebooks/          # Exploratory and diagnostic notebooks
├── tests/              # Unit tests
├── results/            # Output figures and tables
├── paper/              # Manuscript and poster source
└── docs/               # Reference documents (research plan, technical scaffold)
```

## Quick start

```bash
pip install -r requirements.txt
pip install -e src/queuediff
```
